This is Commentary.

Commentary is a web comment system built on App.net.

View a demo.

Commentary runs entirely in the browser. No backend is required, beyond that provided by App.net. To add comments to a page, just add a few lines of JavaScript and an App.net post ID to use as the comment tree root.

Commentary is a work in progress. At the moment, it only supports logging in and out, with no additional functionality.
